关于本站

本站是一个在线星盘程序

目标

目前程序只实现了本命盘和流年盘两中星盘，后续会继续增加功能。

计划功能：法达星限，次限，三限，八字排盘

本站源代码

本站源代码开放，也希望爱好占星的人士能够贡献力量。 

github 地址	
https://github.com/openmartin/astro_render.git

如何在本机运行本程序

本站使用Python语言开发，运行环境Python2.6，Django1.5 

1.安装Python2.6
http://www.python.org/download/releases/2.6.5/
						
2.安装setuptools
https://pypi.python.org/pypi/setuptools/0.6c11

3.然后安装pip
http://www.pip-installer.org/en/latest/installing.html

4.安装依赖包
pip install django==1.5 pytz socialoauth django-celery

6.运行
python manage.py runserver 80

用浏览器打开 http://localhost 就可以看到本站了
